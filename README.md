# Movie Streaming Website

Welcome to the Movie Streaming Website! This project is a Next.js-based web application that allows users to browse, search, and stream their favorite movies.

- **Status**: This project is currently in development. Some features may not be fully functional yet.


## Live Demo

Check out the live demo: [Movie Streaming Website](https://movie-streaming-next-app-pk.vercel.app/)


## Features

- Browse popular and trending movies
- Search for movies by title
- View detailed information about each movie
- Stream movies directly from the platform
- Responsive design for mobile and desktop

## Technologies Used

- **Next.js**: Framework for server-rendered React applications
- **React**: JavaScript library for building user interfaces
- **Tailwind CSS**: Utility-first CSS framework for styling
- **API Integration**: Fetch movie data from a public API (e.g., The Movie Database API)
- **Vercel**: Deployment platform for hosting the website

## Setup and Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/yourusername/movie-streaming-next-app.git
    cd movie-streaming-next-app
    ```

2. Install dependencies:

    ```bash
    npm install
    ```


3. Run the development server:

    ```bash
    npm run dev
    ```

    The application will be available at `http://localhost:3000`.

4. Build the application for production:

    ```bash
    npm run build
    npm start
    ```

## Usage

- Visit the homepage to browse popular and trending movies.
- Use the search bar to find movies by title.
- Click on a movie to view detailed information, including its description, release date, and rating.
- Start streaming the movie directly from the website.

## Project Structure

- `/pages`: Contains all the Next.js pages (e.g., Home, Movie Details)
- `/components`: Reusable UI components (e.g., MovieCard, SearchBar)
- `/styles`: Global and component-specific styles using Tailwind CSS
- `/public`: Static assets (e.g., images, icons)
- `/api`: API routes and external API integrations
